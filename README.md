# sarahcodeswell.com

Personal portfolio site for Sarah Ritter.

## Stack
- Next.js 14
- CSS Modules
- Deployed on Vercel

## Local dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

1. Push to GitHub
2. Import repo at vercel.com/new
3. Add custom domain: `sarahcodeswell.com`
4. Point your domain's DNS to Vercel (add A record and CNAME per Vercel instructions)

No environment variables needed.
